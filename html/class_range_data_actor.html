Detailed Description

RangeDataActor is configured to observe a channel of a dataStorage if the channel value is between lowerbound and upperbound (including the values itself) the gpio pin will change its output the output is defined by inRangeIsOn, if inRangeIsOn==true, the output is HIGH if the channel is in the range and LOW if channel value is out of range if inRangeIsOn==false, the output is LOW if the channel is in the range and HIGH if channel value is out of range
